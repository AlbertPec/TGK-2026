# Van buren

Van buren to gra typu RPG osadzona w świecie inspirowanym klimatem postapokaliptycznym oraz klasycznymi produkcjami cRPG, takimi jak Fallout. Gra prezentowana jest w grafice 2D, w rzucie izometrycznym (lub z perspektywy z góry, podobnej do Stardew Valley).

Gracz ma do dyspozycji różne rodzaje uzbrojenia, które umożliwiają walkę z przeciwnikami w turowym systemie potyczek, przypominającym mechaniki znane z serii X-COM. Rozgrywka koncentruje się na odkrywaniu fabuły poprzez eksplorację świata, starcia taktyczne oraz rozwiązywanie różnorodnych zagadek środowiskowych.

Gra oferuje krótką, spójną historię, którą gracz poznaje stopniowo w trakcie rozgrywki.
